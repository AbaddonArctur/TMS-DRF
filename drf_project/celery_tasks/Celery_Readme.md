
Для локального запуска нужен redis, можно через "docker-compose up -d redis" в терминале.\
Не забываем про "cd drf_project". Далее - "python manage.py runserver".\
В админке выбираем интервалы, создаем интервал с числом периодов "1" и интервальным периодом "минуты" (например). В периодических задачах присваиваем название, выбираем нашу задачу и выбираем период, сохраняем.
### Celery worker
"celery -A drf_project worker -l info" в одном терминале
### Celery beat
"celery -A drf_project beat -l info" в другом
